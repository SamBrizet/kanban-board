# Kanban Board

Kanban Board es una aplicacion Vue 3 para gestionar columnas y tareas con una experiencia inspirada en Notion + Trello. Funciona completamente en frontend y persiste datos en LocalStorage.

## Descripcion

Incluye creacion de columnas, tareas con prioridad y responsable, edicion, eliminacion, filtros, drag and drop entre columnas, modo oscuro y responsive.

## Tecnologias

- Vue 3
- Vite
- Pinia
- Composition API
- ESLint
- Auto Animate
